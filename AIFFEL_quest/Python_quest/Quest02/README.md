- 문제에서 요구하는 기능이 정상적으로 작동하는지?
    - 네, 정상적으로 동작합니다.  

```python
  gen = [text[i] + text[i + 1] for i in range(len(text) - 1)]
  count = 0
  diction = {}
  for g in gen:
    for i in gen:
      if g == i:
        count += 1
    diction[g] = count
    print(g + " = " + str(count))
    count = 0
  print(max(diction.values))
```
